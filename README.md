# Кластеризация стран по социально-экономическим показателям

## Тема проекта
Группировка стран на основе индикаторов благополучия и экономического развития.

## Цель
Выделить однородные кластеры стран, интерпретировать их с точки зрения уровня жизни, социальной поддержки, ВВП и свободы.

## Источник данных
- World Happiness Report 2023
- Признаки: \
`Country name` - Название страны (Тип: категориальный) \
`Ladder score` - Индекс счастья (от 0 до 10) (Тип: числовой) \
`Upperwhisker` - Верхняя граница доверительного интервала для Ladder score (статистическая погрешность). Показывает, насколько может быть выше истинное значение счастья с учётом ошибки измерения. (Тип: числовой) \
`Lowerwhisker` - Нижняя граница доверительного интервала для Ladder score. Показывает, насколько может быть ниже истинное значение счастья. (Тип: числовой) \
`Log GDP per capita` - Логарифм ВВП на душу населения (по паритету покупательной способности, ППС) (Тип: числовой) \
`Social support` - Социальная поддержка - доля населения, которая может рассчитывать на помощь родственников, друзей или коллег в трудной ситуации. (Тип: числовой) \
`Healthy life expectancy` - Ожидаемая продолжительность здоровой жизни (в годах при рождении, скорректированная на инвалидность) (Тип: числовой) \
`Freedom to make life choices` - Свобода принятия жизненно важных решений. (Тип: числовой) \
`Generosity` - Щедрость - объём пожертвований и благотворительной активности (остаток после регрессии на ВВП). (Тип: числовой) \
`Perceptions of corruption` - Восприятие коррупции - степень, в которой жители считают, что коррупция распространена в правительстве и бизнесе. (Тип: числовой) \
`Dystopia + residual` - Антиутопия + остаток. Это поправочный коэффициент, который включает: оценку жизни в гипотетической "антиутопии" (самой несчастной стране); остаточную ошибку модели (часть счастья, не объяснённая другими факторами) \
